Welcome to the Help Page!

Introduction

Learning any new web application can be tricky, the goal of this page is to make the transition as seamless as possible for new users. Below you will find various documentation on how to create a binder, add files to your binder, submit a binder for review as well as other functionalities. Depending on what type of user you are (Applicant, Evaluator or Organizer) the various sections will be very helpful.

Creating a Binder

In order to create a binder, please click on the My Binder tab at the top of the page. You can also do this by clicking here. Once this is done, you will be able to create a new binder, or select from previous binders you have created in past years. For detailed instructions on how to create a binder and what contents needed to be added to a specific binder, you can visit the binder help page by clicking this link

Organizing a Binder

If you are accessing the site as an Organizer you can select various binders that have been submitted here. You can also navigate to the proper page via the tabs above, and selecting workbench. These binders represented submitted faculty binders that are ready to passed on to the proper evaluators within the eRPT site. If there is a binder that you are expecting to be there but is not, it is possible that the binder has not yet been submitted for evaluation by the faculty member who it belongs to.

Evaluating a Binder

If you are accessing the site as an Evaluator you can select various binders that are available for evaluation here. You can also navigate to the proper page via the tabs above by selecting Evaluate. These binders represented submitted faculty binders that are ready to evaluated for various positions. If there is a binder that you are expecting to be there but is not, it is possible that the binder has not yet been submitted for evaluation by the faculty member who it belongs to, or an Organizer has not yet passed it on the all the evaluators that will be viewing the binder.
